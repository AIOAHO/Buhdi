## Getting Started

### Installation and setup

* Clone this repo
* Install the packages: (both within Client and Server branch)
```
npm i
```
* In server branch compile to JS
```
tsc
```
* Run & use Expo (or Expo via Xcode or Web)(both within Client and Server branch)
```
npm start
```
